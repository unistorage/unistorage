Основные термины и понятия
==========================

.. glossary::
    пользователь
        Клиент, имеющий токен для доступа к хранилищу.
        
        :синоним: user

    операция
        Некоторое действие над файлом из хранилища: например, конвертация в другой формат или
        изменение размера картинки.
        
        :синоним: action

    шаблон
        Упорядоченный набор операций, совместимых между собой. Под совместимостью подразумевается
        то, что каждую последующую операцию можно применить к результату предыдущей.
        
        :синоним: template
        
    тип
        Каждый файл в хранилище имеет определённый тип контента (MIME-type).
        
        :синоним: type
        
    семейство типов
        Семейство типов -- некоторое подмножество типов. Например, семейство типов ``image``
        включает в себя все типы, начинающиеся со строки ``image/``.

        :синоним: type family

    обычный файл
        Файл, залитый пользователем, либо полученный в результате успешно применённых операций.

        :синоним: regular file

    временный файл
        Пустой файл, находящийся в хранилище главным образом с целью "бронирования" идентификатора
        на время, пока заказанные пользователем операции ждут в очереди или исполняются.

        :синоним: pending file

    идентификатор контента
        Строка, которую пользователь может указать при загрузке файла. Используется для сбора
        статистики.

        :синоним: type id
